# Usage

Navigation: [README](README.md) | [Next: Development](02-development.md)

The module handles:

- `md`;
- `markdown`;
- `mdown`;
- `mkd`;
- `mmd`;
- `mermaid`.

Stable routes include:

```text
babelchrome://markdown/file/<encoded-path>
babelchrome://markdown/url/<encoded-url>
```

The viewer supports source-file context actions, local file auto-refresh, selectable themes, a generated table of contents, Mermaid rendering, and the shared `Open with` header control.
