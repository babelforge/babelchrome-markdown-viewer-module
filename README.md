# Markdown Viewer Module

`babelforge.markdown-viewer` is an installable BabelChrome viewer module for Markdown, Markdown-like files, and Mermaid documents.

It renders local and remote sources through Symfony, `league/commonmark`, Twig templates, AssetMapper assets, and `babelforge/babel-chrome-viewer-kit`.
